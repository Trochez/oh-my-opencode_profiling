Clone the Repository\n```bash\n# Create a dedicated directory\nmkdir -p ~/opencode-profile-tool && cd ~/opencode-profile-tool\n\n# Clone the target repository\ngit clone git@github.com:Trochez/oh-my-opencode_profiling.git .\n```\n\n### 2. Verify Repository Structure\n```bash\nls -la\n# Should show:\n# .sisyphus/\n#   .sisyphus/plans/opencode-profile-tool.md\n#   tasks/\n#   workers/\n# .omx/\n#   state/\n# README.md\n# .env.example\n```\n\n## 🛠️ Environment Configuration\n\n### 1. Create Configuration Files\n```bash\n# Copy example environment file\ncp .env.example .env\n\n# Generate a secure secret key (Linux/macOS)\nopenssl rand -hex 32 > .secret-key\n\n# On Windows PowerShell\n# $key = [System.Text.Encoding]::UTF8.GetString([System.Security.Cryptography.RandomNumberGenerator]::Create().GetBytes(32))\n# $key | Out-File -FilePath .secret-key -Encoding ascii\n```\n\n### 2. Configure Environment Variables\nEdit `.env` with your preferred editor:\n```bash\ncode .env\n```\n\nSet these required variables:\n```env\n# Database configuration\nDATABASE_URL=sqlite://./dev.db\n\n# Server configuration\nAPI_PORT=3001\nHOST=0.0.0.0\n\n# Authentication\nAUTH_TOKEN_SECRET=your-generated-secret-key\nAUTH_TOKEN_EXPIRES_IN=86400  # 24 hours in seconds\n\n# Feature toggles\nENABLE_AUTH=true\nENABLE_METRICS_COLLECTION=true\n```\n\n## 📦 Dependency Installation\n\n### Using pnpm (Recommended)\n```bash\n# Install all dependencies\npnpm install\n\n# Install additional development dependencies\npnpm add -D typescript ts-node @types/node\n```\n\n### Using npm (Alternative)\n```bash\nnpm install\n```\n\n## 🏗️ Build Process\n\n### 1.
